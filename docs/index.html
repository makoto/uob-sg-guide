Queenstown Liveability Study

Geospatial analysis of perceived liveability in Singapore's Queenstown district.

3D Buildings Viewer

8,671 extruded buildings with height-based colour coding, 26 toggleable map layers (12 categories) with provenance filter (Gov.sg / Global), and a choropleth heatmap with 25 subzone-level metrics including remote sensing indicators (NDVI, LST, canopy cover) and BEH-NWI walkability scores. Built with CesiumJS and 3D Tiles.

Data sources: OpenStreetMap building footprints (Overpass API) enriched with HDB property records (data.gov.sg). 308 HDB blocks matched by block number + street name.

Height range: 3m – 183m (mean 14.2m, median 10m). Tallest: mTower (183m), SkyVille @ Dawson (141m, 47 storeys).

Click any building to see: name, type, height, storeys, data source, and HDB year completed.

HDB blocks layer: 308 HDB dots coloured by construction era — red (pre-1980), orange (1980–1999), green (2000+). Click shows year completed, dwelling units, and use flags.

Height control zones: Semi-transparent URA polygons showing storey limits from the Master Plan.

Street network layer: 3,788 road edges from OSMnx drive network, coloured by betweenness centrality to highlight key corridors.

Remote sensing layers: 4 pixel-level raster overlays (NDVI, LST, NDBI, canopy cover) rendered via CesiumJS SingleTileImageryProvider, plus a clickable 100 m grid (2,402 cells) coloured by NDVI with per-cell NDVI, NDBI, LST, and canopy % on click. The original subzone-level remote sensing GeoJSON (15 polygons) remains for choropleth use.

Mapillary images layer: 5,000 sampled street-level image locations from Mapillary/ZenSVI.

Choropleth heatmap: Select from 25 metrics in the Layers panel dropdown to colour subzones — including population density, elderly share, amenity density, MRT stations, cycling paths, park connectors, resale flat price, avg building height, HDB blocks, total buildings, max building height, resale transactions, avg HDB year built, dwelling units, dwelling density, plus 6 remote sensing indicators: Vegetation (NDVI), Built-up (NDBI), Land Surface Temp, Tree canopy cover, GHSL building height, and Surface elevation, plus 4 walkability metrics: Intersection density, Transit access, Destination access, and Walkability (BEH-NWI). Uses a 5-step YlOrRd colour ramp with dynamic legend.

Data Catalogue

Complete inventory of 60+ datasets from 8 sources (data.gov.sg, URA, OneMap, LTA DataMall, Mapillary, DMP25, OSMnx, Google Earth Engine) with access requirements and spatial join fields.

Tech Catalogue

Technologies, libraries, and frameworks: Python geospatial stack (Shapely, OSMnx, GeoPandas, GEE), CesiumJS 3D viewer, 3D Tiles pipeline (py3dtiles, mapbox_earcut), ZenSVI/Mapillary, and GitHub Actions CI/CD.
