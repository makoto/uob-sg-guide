Singapore District Liveability Study Data Guide

Geospatial analysis of perceived liveability across five Singapore planning districts: Queenstown, Marymount (Bishan), Outram, Bukit Merah, and Newton.

Disclaimer: This is an AI-assisted project. The data and analysis have not been independently verified — please cross-check with original sources before relying on them.

Map Viewer

Multi-district viewer with district selector (Queenstown, Marymount (Bishan), Outram, Bukit Merah, Newton). Extruded 3D buildings for all 5 districts with height-based colour coding, 29 toggleable overlays in 10 collapsible groups with feature counts per layer, provenance filter (Gov.sg / Global), and a categorized choropleth accordion with 33 subzone-level metrics across 5 sub-categories (with type badges: CNT/RATE/IDX/SAT). Includes an in-viewer Data Catalogue modal with live layer and metric tables. Shareable URL state — use the “Copy link” button to share your exact view (district, layers, subzone filter, choropleth, basemap, RS grid mode). Built with deck.gl + MapLibre GL JS. GPU-accelerated rendering, ~700 KB bundle, free CARTO basemaps (Dark, Light, Voyager). No API token needed.

Map Viewer User Manual

Guide to the deck.gl viewer: map navigation, district selector, 29 toggleable layers across 10 groups, 33 choropleth metrics, RS grid colour modes, shareable URL state with Copy link button, click popups, and troubleshooting tips. Includes 9 annotated screenshots showing each major UI component.

Data Catalogue

Complete inventory of 60+ datasets from 7 sources (data.gov.sg, URA, OneMap, LTA DataMall, DMP25, OSMnx, Google Earth Engine) with access requirements and spatial join fields. Includes a Demographics Matrix table showing area, population, density, elderly, and dwelling units for all 45 subzones. Also accessible as a live summary modal from within the map viewer.

Tech Catalogue

Technologies, libraries, and frameworks: Python geospatial stack (Shapely, OSMnx, GeoPandas, GEE), deck.gl + MapLibre GL JS viewer, and GitHub Actions CI/CD.

Glossary & Data Dictionary

Definitions of acronyms and technical terms: DEM, NDVI, LST, BEH-NWI, Tobler’s hiking function, HDB, URA, MRT, choropleth, z-score, and more.
